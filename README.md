# RIPRENDI.TI

L'app che ti ridà i soldi che non sapevi di avere.

## Categorie rimborso
- Treni (ritardi, cancellazioni)
- Voli (EU261: ritardi, overbooking, cancellazioni)
- Amazon (ritardi Prime, consegne)
- Banche (commissioni abusive)
- Assicurazioni (mancati rimborsi)
- Bollette (doppie fatturazioni, penali)
- PA (multe, ticket, IMU, TARI)

Tecnologia: Node.js + Express
